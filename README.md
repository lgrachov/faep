## Contributing

Contributions are always welcome!

You can contribute by going to the FAEP Figma file & copying it and sharing!
## Installation

You must have the requirements complete:
 1. 64GB SSD
 2. Video Card
 3. Figma Desktop

Then, you can run the `[BOOTSECTOR]` flow.
